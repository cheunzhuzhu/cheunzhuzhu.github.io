我的博客日记

今天终于完成了我的博客模板的基本排版，接下来就给大伙分享一下我搭建博客的时候遇到的一些问题吧。

博客搭建

今天终于完成了我的博客模板的基本排版，接下来就给大伙分享一下的搭建博客的时候遇到的一些问题吧。

1、准备个人博客需要的工具

安装：Git

Git是用来创建博客文档、上传到Github、安装插件都需要的工具。

安装：Node.js

安装Node.js完成后，打开git-hash.exe输入:

$ node -v && npm -v

检测是否安装成功（显示其版本号）如：

如果没有显示版本号就是安装失败就再安装一次。

2、如何创建一个个人博客

安装Hexo

打开git-bash.exe创建一个文件夹:

$ mkdir blog

进入文件夹:

$ cd blog

安装hexo-cli插件:

$ npm install -g hexo-cli

查看版本:

$hexo -v

初始化hexo

$ hexo init myblog

进入myblog

$ cd myblog

$ npm install

新建完成后在myblog目录下打开git-bash.exe输入 ls

会显示node_modules、public、scaffolds、source、themes、_config.yml 6个文件夹，没有出现的话就是出错了，建议重新再创

在myblog目录下打开git-bash.exe

$ hexo clean && hexo g && hexo s

清理hexo缓存，更新上传，创建本地hexo服务

然后在浏览器地址栏输入：localhost:4000 就可以看到刚刚新建的博客了，不过只能让自己看见

3、如何创建一个大家都可以看见的博客

只有自己能看见自己的博客是不是感觉到很没意思，那就开始搭建一个大家都能看见的博客吧

创建Github个人仓库

首先第一步是拥有一个Github的账户

登录Github新建仓库（必须要和用户名一样不然后面容易出现问题）

生成SSH密钥添加到Github

在git-bash中输入

绑定Github

$ git config --global user.name "Github的用户名"

$ git config --global user.email "Github注册输入的邮箱"

创建SSH密钥

$ ssh-keygen -t rsa -C "Github注册的输入的邮箱"

然后就会在C://用户/用户名 下生成一个.ssh文件夹，打开文件夹里的id_rsa.pub文件，里面就是你的公共密钥（如果找不到pub文件可以试试打开拓展名，反正我没打开的时候找不到/(ㄒoㄒ)/~~）

返回Github的setting中，找到SSH keys设置，点击New SSH key，把你的公共密钥复制进去。

返回git-bash

$ ssh -T git@github.com

修改站点的配置文件_config.yml

在配置文件的最底下输入

注: yml配置文件的格式要求特别养个，所以格式一定要对常见的就会有：后面没有加一个空格导致报错，所以还是不要瞎捣鼓，报错了就麻烦了，我都被烦死了/(ㄒoㄒ)/~~

安装deploy-git

$ npm install hexo-deployer-git --save

清理缓存、更新文件、上传部署文件

$ hexo clean && hexo g && hexo d

这样就完成了把你的博客部署到Github服务器上了

在浏览器地址栏输入 http://你的Git用户名.github.io就可以看到你的博客咯

4、个人域名

觉得你现在的网站太难记了怎么办，当然是设置一个个人域名啦！

可以选择阿里云或者是腾讯云之类的买个自定义域名

购买成功后

点击你的域名-全部域名-解析-添加记录

记录类型：CNAME

主机记录：@

解析线路：默认

记录值：yourname.github.io

确认

在添加一条把 @ 改成 www

然后在Github选择刚刚创建的仓库-settings-pages设置Custom domain输入你购买的域名

然后！！然后！在你blog文件中的source新建一个名为CNAME的文件，记得删掉后缀，在文件里面写上你刚买的域名

最后

$ hexo clean && hexo g && hexo d

然后就可以在浏览器输入你自己购买的域名看到你的博客啦

然后就可以开始写文章，在git-bash中输入

$ hexo new 文章标题

会在source/_post文件中生成一个后缀是.md的文件，这个时候你就可以使用Typora编辑器编辑了（当然记事本也可以，只是Typora比较的方便)

写完后千万不要忘记更新部署

$ hexo clean && hexo g && hexo d

有时候会出现hexo d传不上去的情况，可能是网络不好，Ctrl+C退出在输入多几次就行了

或者是

$ rm -rf .deploy_git 和 $ git config --global core.autocrlf false

之后再$ hexo clean && hexo g && hexo d

hexo的基本配置都是在_config.yml中更改

好了就是这样的，我能成功安装博客还离不开大佬的帮助.

更改hexo主题

由于我使用的yelee的主题，所以就用这个主题演示一下安装和遇到的一些问题

刚开始安装主题的时候发现主题自带有说明只需要在git-bash里输入

$ git clone https://github.com/MOxFIVE/hexo-theme-yelee.git themes/yelee

但是我发现emmmm，不行众所周知GitHub是国外的网站，访问他有点点的复杂（就是科学上网）所以我选择的是直接下载压缩包

步骤就是点击网址里的Code在下拉菜单的最后一个就是下载压缩包Download ZIP，然后解压文件夹里的yelee文件夹到myublog的themes文件夹里就行了。
